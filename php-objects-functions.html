PHP Objetos y Funciones


La programación se vuelve tediosa cuando tenemos que reescribir un número de veces porciones muy similares de código, para eso están las funciones y los objetos. Las funciones son un conjunto de instrucciones que actúan implementando alguna función en el programa. Cada vez que necesitamos ejecutar esas instrucciones invocamos el nombre de la función. PHP ostenta una gran cantidad de funciones precompiladas como la función:

La característica de una función es que usa paréntises para pasarle los argumentos a la función, sin embargo print es una pseudo-función que muestra texto en la pantalla y es la única función que no requiere paréntesis.
